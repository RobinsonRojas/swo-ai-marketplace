# Web automate testing

Sigue las instrucciones detalladas a continuación para crear o actualizar scripts de automatización de pruebas web utilizando Playwright.

## Creación del proyecto

Si el usuario de solicita crear un nuevo proyecto de automatización de pruebas web, sigue las siguientes instrucciones:

1. Crea una nueva carpeta en la raíz del proyecto con el nombre `web_automate_testing`.
2. Dentro de la carpeta `web_automate_testing`, crea un nuevo proyecto de Playwright utilizando el comando `npx playwright init` y npx `playwright install`
3. crea dentro de la carpeta `web_automate_testing` una nueva carpeta llamada `tests` donde se almacenarán los scripts de automatización de pruebas web.
4. Informa al usuario sobre la ubicación del proyecto creado y los pasos para ejecutar las pruebas

## Adición o actualización de scripts de automatización de pruebas web

Si el usuario solicita agregar o actualizar scripts de automatización de pruebas web, sigue las siguientes instrucciones:

1. Solicita al usuario el nombre del script de automatización de pruebas web que desea crear o actualizar.
2. Verifica si el script ya existe en la carpeta `web_automate_testing\tests`.
